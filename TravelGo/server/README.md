`server`
========
This repository contains materials for *TravelGo* backend development.

## Navigating

### Testing
- To test the backend, run `node server.js`

---

### Setting up
- To set up the backend on your local computer on VSCode, run `npm install <package name>` in terminal
  - `express` 
  - `mongoose` 
  - `cors` 
  - `dotenv`
  - `react-scripts`
  - `bcryptjs`
  - `jsonwebtoken`
  - `--save-dev nodemon`
  - `axios`
